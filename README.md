# Cyber Survivor

## RUS
2D-рогалик-шутер в стиле Vampire Survivors. Выживайте среди волн врагов, прокачивайтесь, выбирайте перки и открывайте скины.

### Управление
- WASD — движение  
- Мышь — прицел  
- Space — рывок  
- Tab — вкл/выкл автострельбу  
- Q — активная способность  
- Esc — пауза/меню


## ENG
2D roguelike survival shooter inspired by Vampire Survivors. Survive enemy waves, level up, pick perks, unlock skins.

### Controls
- WASD — move  
- Mouse — aim  
- Space — dash  
- Tab — toggle auto-fire  
- Q — active ability  
- Esc — pause/menu
